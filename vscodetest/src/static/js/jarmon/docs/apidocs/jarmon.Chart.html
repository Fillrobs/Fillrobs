Class jarmon.Chart

A class for creating a Flot chart from a series of RRD Queries

Constructor

jarmon.Chart ( template , options )
Parameters:
template <Object> A jQuery containing a single element into which the chart will be drawn
options <Object> Flot options which control how the chart should be drawn.

Methods

addData

void addData ( label , db , enabled )
Add details of a remote RRD data source whose data will be added to this chart.
Parameters:
label <String> The label for this data which will be shown in the chart legend
db <String> The url of the remote RRD database
enabled <Boolean> true if you want this data plotted on the chart, false if not.
Returns: void

draw

Object draw ( )
Draw the chart A 'chart_loading' event is triggered before the data is requested A 'chart_loaded' event is triggered when the chart has been drawn
Returns: Object
A Deferred which calls back with the chart data when the chart has been rendered.

endTime <Number> The end timestamp
Returns: void

void switchDataEnabled ( label )
Enable / Disable a single data source
Parameters:
