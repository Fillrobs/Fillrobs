Class jarmon.RrdQuery

A wrapper around an instance of javascriptrrd.RRDFile which provides a convenient way to query the RRDFile based on time range, RRD data source (DS) and RRD consolidation function (CF).

Constructor

jarmon.RrdQuery ( rrd , unit , transformer )
Parameters:
rrd <Object> A javascriptrrd.RRDFile
unit <String> The unit symbol for this data series
transformer <Function> A callable which performs a tranfsformation of the values returned from the RRD file.

Methods

getData

Object getData ( startTimeJs , endTimeJs , dsId , cfName )
Generate a Flot compatible data object containing rows between start and end time. The rows are taken from the first RRA whose data spans the requested time range.
Parameters:
startTimeJs <Number> start timestamp in microseconds
endTimeJs <Number> end timestamp in microseconds
dsId <Variant> identifier of the RRD datasource (string or number)
cfName <String> The name of an RRD consolidation function (CF) eg AVERAGE, MIN, MAX
Returns: Object
A Flot compatible data series eg label: '', data: [], unit: ''

Array getDSNames ( )
Return a list of RRD Data Source names
Returns: Array
An array of DS names.
