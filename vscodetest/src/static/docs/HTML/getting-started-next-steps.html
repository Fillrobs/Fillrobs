Next Steps

Once CloudBolt is installed and customized, there are many more features to explore. Get started with these ideas:



Global search

  • Use the Global search to navigate around the CloudBolt UI. There is an Admin-specific search on the Admin page that displays results from an Admin-user perspective. 


CloudBolt configuration

  • Check and potentially change the configuration of your CloudBolt instance from Admin > System > Miscellaneous Settings.


Manage Preconfigurations

  • Manage Preconfigurations (server size, software stack, etc.) from Admin > CloudBolt Objects > Preconfigurations. 


Set up Rates

  • Learn about and set up Rates from Admin > CloudBolt Objects > Rates.


Explore Resource Handlers

  • View, add, and configure Resource Handlers. Click Admin > Resource Handlers. Add Resource Handlers for the technology in your IT shop.


Server change tracking

  • Try changing a VM attribute, such as memory, from the Resource Handler's own UI. Click Refresh Info on the server in CloudBolt. Click the server's History tab to see the changes. 


Group tree

  • Explore the possibilities with CloudBolt's Group tree. 
  • Set up more Groups and Group types.
  • Set quota limits and look at the quota usage view of the Group tree.
  • Delegated Group Administration: Set another user as the Group Admin, then remove your user from the Group.
  • Manage User Permissions: Set up a user who is a Requestor only, to explore what they can see and do.


Branded portals

  • Upload a new logo to include your own company's branding in Admin > System > Branded Portals.


Single Sign On

  • Check out our External Authentication options for increased security.


Enterprise functionality

  • Contact us to explore enterprise functionality beyond what's found in the documentation, if you're interested in support for provision engines, configuration management systems, network virtualization, etc. 


Job Runtime Triggers

  • Explore the orchestration framework and actions to integrate with other systems during the provisioning process.
