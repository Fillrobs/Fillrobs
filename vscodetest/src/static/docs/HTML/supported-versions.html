Support Lifecycle Policy

General Availability

The date the version is released and support starts.

Deprecation

This date is typically 12 months from General Availability.

Upgrades from this version to the latest version will stop being tested on this date. Upgrades are still supported, but may take multiple steps.

End of General Support

This date is typically 3 months from Deprecation.

CloudBolt Professional Services can still be engaged to help after this date, but CloudBolt support will require you to upgrade to a newer version before troubleshooting/advising.
