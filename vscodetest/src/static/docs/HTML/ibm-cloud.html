IBM Cloud

The IBM Cloud Resource Handler allows users to integrate with IBM Cloud and access their IBM Cloud data from within CloudBolt.

The IBM Cloud account you're connecting to must be at minimum a "Pay-as-you-go" account.
To set up the IBM Cloud Resource Handler:
  1. Go to Admin > Resource Handlers.
  2. Click +Add a new resource handler…
  3. Select IBM Cloud. 
  1. Find your Classic Infrastructure API Key and enter the API Username (the Account ID) and API Key. See Managing user API keys in the IBM Cloud docs for more information on creating and finding your API Keys.

Network Configuration

After importing datacenters and networks, add the imported networks to their corresponding datacenters.

  1. On the Datacenters tab, click on an Environment.
  2. Click the Networks tab, then Make Changes. When configuring an IBM Cloud environment's networks, ensure there is at least one PRIVATE network.
