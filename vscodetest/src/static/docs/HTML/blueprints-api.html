blueprints / {id}

Represents a Blueprint in CloudBolt.

Endpoints

/api/v3/cmp/blueprints/{id}/

MethodEndpoint NameDescription


/blueprints/
GET Returns a paginated list of all Blueprints in CloudBolt.
POST Create a Blueprint and returns the newly created Blueprint.

/blueprints/{id}/

GET Returns a specific Blueprint by Global ID.

DELETE Removes a specific Blueprint by Global ID.

/blueprints/{id}/export/
Exports the Blueprint with the given ID.
/blueprints/{id}/deploymentSchema/
Generates a deployment schema for the given Group and Environment.

/blueprints/{id}/deploy/
Generates and submits an Order to deploy an instance of this Blueprint.

/blueprints/{id}/samplePayload/
Generates a sample deployment payload for the given Group and Server item Environments.

/api/v3/cmp/blueprintCategories/{id}/

MethodEndpoint NameDescription

/blueprintCategories/
GET Returns a paginated list of all Blueprint Categories in CloudBolt.
POST Creates a Blueprint Category and returns the newly created Blueprint Category item.



/blueprintCategories/{id}/

GET Get a specific Blueprint Category by Global ID.
PATCH Update a specific Blueprint Category and return the updated Category by Global ID.
/blueprintCategories/{id}/
Delete a specific Blueprint Category.
