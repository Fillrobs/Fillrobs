VMWare Cloud on AWS

The VMWare Cloud on AWS Resource Handler allows users to use VMWare on AWS rather than hosting VMWare locally.

To set up this Resource Handler:

  1. Go to Admin > Resource Handlers
  2. Click + Add a resource handler…
  3. Select VMWare on AWS.
  1. Enter the login credentials for a Software Defined Datacenter (SDDC).
  2. The API token field information comes from the account that controls these SDDCs.
