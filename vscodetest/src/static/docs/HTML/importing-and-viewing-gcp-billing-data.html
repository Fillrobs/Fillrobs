Importing and Viewing GCP Billing Data

In CloudBolt, you can import your GCP billing data and view monthly billed costs for individual GCP projects.

Set Up a Single GCP Project

Required steps:

  1. Enable the export of your GCP billing data to BigQuery. Refer to these instructions for Setting Up the GCP Billing Data export to BigQuery.
  2. After creating your billing export dataset in BigQuery, take note of the name of the table in your dataset.
  3. In the GCP console, go to the Service accounts page and select the project where your BigQuery dataset is stored.
  4. Create a new Service account to access your billing dataset.
    • When the account is created, a JSON file containing a credentials key is generated and downloaded to your computer. Save this file in a safe location; once downloaded, it cannot be redownloaded.
  5. Go to the IAM page in the GCP console and edit the permissions for the Service account you just created. Make sure to give it the BigQuery Admin Role.
  6. In CloudBolt, go to the details page for a GCP Project for which you want to track billing.
  7. Click Edit.
  8. Upload the Key file you downloaded for the Service account created in step 4, and add the Billing table name you noted in step 2.
  9. Click Save.

Set up Multiple GCP Projects

The following steps are required for each additional GCP project for which you want to track billing:

  1. In the GCP console, copy the email address of the Service account you created to access the billing dataset.
  2. Go to the IAM page for the project you want to add.
  3. Click Add.
  4. In the New members textbox, add the email address from the previous step.
  5. Select the BigQuery Admin Role > Save.
  6. In CloudBolt, go to the details page for the GCP project you want to add.
  7. In the Edit dialog, add the same Key file and Billing table name you used for the original project.
  8. Click Save.

Viewing

You can view billing summaries from GCP projects’ and Resource Handlers’ Billing tabs. Project Billing tabs will display graphs with data aggregated per-project. Resource Handler Billing tabs will display graphs with data aggregated across multiple projects.
