Data Protection Plan Parameters

Data Protection Plans can be applied to Environments or Servers as Parameters. When a Data Protection Plan Parameter is added to an Environment, any servers in that Environment will have the ability to use a Data Protection Plan.

Add a Data Protection Parameter to an Environment

  1. Navigate to an Environment with Servers that need data protection.
  2. Click the Parameters tab. 
  3. Click +Add and select Data Protection Plan. The Data Protection Plan parameter is now available for Servers within this Environment. Repeat this process for any additional Environments.
  4. Click X in the Actions column to remove the Parameter and associated plans from the Environment.

Provision a Server with a Protection Plan

You can predefine the Data Protection Plan for a Server at order time, or let the user choose on ordering.

Predefine a Data Protection Plan

  1. In CloudBolt, create a Blueprint or Edit an existing Blueprint.
    1. If editing an existing Blueprint, click Build tab > Configured Environments > Edit the Environment.
  2. In the "Blueprint Item Parameters for [Your Environment]" form, click the Data Protection Plan toggle to Predefine and select a plan from the dropdown.
  3. Click Save.
  4. On the Overview tab, click Order.
Parameters that have a single value provided by the Environment will not be shown to users at order time; the value will simply be included once the Blueprint is ordered.

Add and Remove Data Protection Plans

On an Environment

  1. In the navigation menu, click Resources.
  2. ​Order a new server and select the Environment that contains the Data Protection Plan parameter.
  3. Choose an option from the Data Protection Plan dropdown. The new server will have its data protected with the chosen plan.

On a Server

  1. Navigate to the desired server. The server chosen must be in an Environment that has the Data Protection Plan parameter added.
  2. ​Click the Parameters tab.
  3. Click +Add, and select Data Protection Plan.
  4. Click Next, and select a value parameter (plan option) from the dropdown.
  5. Click Save.

View the protection plan:

  1. Click the Details tab on the server to view the chosen Data Protection Plan.
  2. Click the Parameters tab. Click the pencil icon to change the selected Data Protection Plan option for the server.
  3. Click X in the Actions column to remove the parameter and associated plans from the Environment.
The Manage Data Protection Plan permission is required to manage servers' data protection plans via the parameter.
