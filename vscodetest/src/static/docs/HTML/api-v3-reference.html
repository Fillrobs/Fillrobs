API v3 Reference Guide

The CloudBolt API uses {id} and {resourceID} as parameters to return specific items from a collection. This value is the Global ID assigned to CloudBolt resources.

This is a full list of the currently available resources and their descriptions.

All API v3 endpoints are prefixed with /api/v3/cmp/, for example, /api/v3/cmp/brandedPortals/.
