Servers

  1. In the CloudBolt navigation bar, click Resources > Servers.
  2. Click New Server, or use the Server List to view existing Servers.

Customize the Order Form to change what options are available during the order process.

Server Summary

View Server Summary information in the top bar of a Servers page. The statistics that display here are dependent on the permissions of the logged-on user.

  • Power Status: View Power Status for the Servers the user has access to in a pie chart.
  • Installed OS: View Number of Servers with an indicated OS.
  • Jobs: View Jobs that are in a Complete state (Success, Warning, and Failure)
  • Server Status: View the total number of Servers broken down by status.
  • Resource Handlers: View the Resource Handlers being used by the Servers.
  • Owners: View a pie chart of all the Owners of each Server.
  • Groups: View a summary of the Groups belonging to the Servers.
  • Environments: View a summary of the Environments belonging to the Servers.
