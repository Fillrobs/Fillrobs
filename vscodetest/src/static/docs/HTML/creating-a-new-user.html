Creating a New User

Only an Admin user may create users in CloudBolt.

  1. Click Admin > Users.
  2. Click + Create a new User.
  3. Enter user information for all the required fields.
  4. Click Create.

Security Challenge Question

The first time a user logs in to their account, they will be asked to choose a security challenge question and provide an answer. This answer will be used when a password reset is initiated. See the Passwords section below. The answer to the challenge question is not case sensitive. 

Disable the Security Challenge Question in Miscellaneous Settings.

Email Verification

When an existing user’s email address is changed, a verification email with a link is sent to the corresponding address. When the user clicks this link, the email address is saved in the database and marked as verified and active. Email Verification is Enabled by default.

Enable/Disable Email Verification:
  1. Click Admin > Miscellaneous Settings.
  2. Toggle Require User Email Verification to Disable or Enable.

Passwords

When a new user is created by an Admin, passwords can be assigned in one of two ways:

  • A temporary password is assigned to the User by the Admin. This is the default behavior.
  • An email is sent to the User with a temporary password. The User will log in with the temporary password, then change their password from the UI. Accomplish this using the "Require New Users to Self-Assign Passwords" Miscellaneous Setting.                            

Change a Password

After a new user is created, they may change their temporary password to a password of their choosing. Admin users may also change another user’s password.

  1. Click user profile (non-admin) or click Admin > Users and select a user.
  2. Click Manage profile.
  3. In the Change password and Confirm password fields, enter the new password.
  4. Enter the old password in the Original Password field if required. See Original Password Verification in Miscellaneous Settings.

Password Reset

When a password reset is requested, the user must answer a password reset challenge question. This question is set the first time a user logs in to their account.

  1. On the CloudBolt login screen, click Forgot your password?
  2. Enter your email address. An email will be sent that contains a recovery link to reset your password.
  3. Click the link in the email, and enter the answer to the password reset challenge question.
  4. Enter a new password and re-enter in the provided fields.
  5. Click Save password.
If "Email default from address" is not set up in Email settings, the password reset email may go to the user’s spam folder.
