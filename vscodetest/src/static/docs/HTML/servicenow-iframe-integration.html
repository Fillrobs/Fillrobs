Embed CloudBolt in ServiceNow (IFrame)

Learn how to embed CloudBolt inside your ServiceNow portal.

  • Considerations

  • Using Dashboards

Considerations

  • CloudBolt must have a trusted SSL certificate in place. You can check this by looking for a locked padlock icon to the left of the CloudBolt instance URL.
  • This setup assumes you have an existing ServiceNow account and can log in using an Administrator role.
  • For best user experience, enable CloudBolt automatic SSO login to ensure the CloudBolt login prompt is not shown.

Setup Overview

  1. Set up allowed embed URLs in CloudBolt.
  2. Set up IFrame in ServiceNow using Homepage or Dashboards.
  3. Configure the IFrame in ServiceNow.

Set up Embed URLs in CloudBolt

  1. In CloudBolt, navigate to Admin > System > Miscellaneous Settings.
  2. Scroll down to Embed URLs / Frame Ancestors.
  3. Enter the URL of your ServiceNow instance in the Embed URLs field. It will look something like https://yourcompany.service-now.com.


  4. Click Save Changes.
  5. If you have a single CloudBolt appliance skip to step 7.
  6. If you are running multiple CloudBolt appliances, SSH to each web server and run /opt/cloudbolt/manage.py generate_webserver_conf
  7. Navigate to the page in CloudBolt you wish to embed in ServiceNow. A typical use case would be the Catalog page.
  8. In the URL address bar, append ?frame=true to the end of the page's URL and press Enter. Example: https://cloudboltinstance/catalog/?frame=true 
  9. Copy this URL to use in the Set up IFrame in ServiceNow section next.


Embedded Mode

(Optional): To limit certain Users to only access pages with the ?frame=true query string, toggle Embedded Mode to Enabled in Miscellaneous Settings and choose the allowed Global Role(s) .

Exit Embedded Mode

In CloudBolt, change ?frame=true to ?frame=false in the URL to return to CloudBolt with the navigation menu.

Set up IFrame in ServiceNow

Add CloudBolt as an IFrame content block to your ServiceNow instance. Either use Homepage or Dashboards.

The recommended approach is Dashboards, which offers users a streamlined experience. If your version of ServiceNow does not support Dashboards, use Homepage.

Using Dashboards

Create a New Dashboard

  1. Navigate to your ServiceNow instance, logged in as Administrator.
  2. On the left navigation menu, click Dashboards.
  3. Click New.
  4. Enter a Name (required) and any other desired details.
  5. Click Submit.

Add Widgets to the Dashboard

When a Dashboard is created, the Add Widgets menu will appear on the right side of the frame.

  1. Under Widget Category, select Content Blocks.
  2. Select *New IFrame.
  3. Click Add.
  4. Click Click here to configure the IFrame block.
  5. Enter a Name (required).
  6. Paste in the URL copied from CloudBolt in Set up IFrame in CloudBolt, step 7.
  7. Set Sizing to Expand to Fit.
  8. Click Submit.


The CloudBolt page will now appear embedded into your ServiceNow instance.

Dashboard Tips:
  • When your ServiceNow Dashboard is created, click the Widget to open it for further customization.
  • Click the menu to add this to Favorites, Create Tabs to add more pages of CloudBolt to this Dashboard, and click Dashboard Properties to edit the name and tabs of this Dashboard.
  • Click the Configuration icon to set a Dashboard background color to match CloudBolt, currently #ecf0f1

Using a Homepage

If your ServiceNow version does not support Dashboards, set up the CloudBolt IFrame integration using a Homepage. You can convert a Homepage to a Dashboard later if desired.

Create a new Homepage

  1. Navigate to your ServiceNow instance, logged in as an Administrator.
  2. From the Home page, click + to add a new page.
  3. Remove the widget that is automatically added and click Add content.
  4. In the first list, click Content Blocks. In the second list, select New IFrame.
  5. Click the dark gray Add here button.
  6. Click the X to close the window.


Configure the New IFrame

  1. In the new IFrame widget, click Click here to configure the IFrame block.
  2. Enter a Name (required).
  3. Paste in the URL gathered from Set up IFrame in CloudBolt, step 7.
  4. Set Sizing to Expand to fit.
  5. Click Submit.

The CloudBolt page will now appear embedded into your ServiceNow instance.
