CloudBolt Permissions

Learn about the Permissions that exist in CloudBolt, which Roles they are attached to by default, and descriptions of the functionality they allow in CloudBolt.

Overview

Permissions in CloudBolt are assigned to Roles. Each Role comes with a set of default permissions already assigned. Permissions can be added or removed from Roles. 

View Permissions in CloudBolt

  1. In CloudBolt, click Admin > Security > Permissions.
  2. Use the Search box to find specific Permissions, or use the page arrows.
Name Label Description
Orders
order.approve Approve Orders Allows the user to approve and deny Orders.
order.view View Orders Grants the user view-only permission for Orders.
order.submit Submit Orders Allows the user to order Servers and Blueprints.
order.change_attributes Change Order Attributes Change the Order name.
order.choose_recipient Choose Recipient for Orders Allows the user to choose a recipient other than themselves to own the Server(s) and/or Resource(s) created by an Order.
Servers
server.view View Servers Grants the user view-only permission for Servers.
server.control_power Control Power on Servers Power on, power off, reboot, and pause servers.
server.manage_snapshots Manage Snapshots Create, delete, and restore from Server snapshots.
server.manage_nics Manage NICs
server.change_resources Change Resources Change CPU and memory on VMs.
server.console Console
server.remote_terminal Remote Terminal SSH/RDP access to Servers from the CloudBolt UI.
server.manage_applications Manage Apps [Un] Install config manager applications on servers.
server.change_attributes Change Attributes Change Environment, Group, OS Family, and OS Build associates for Servers.
server.manage_labels Manage Labels Add and remove labels on Servers.
server.manager_parameters Manage Server Parameters Add, remove, and modify parameters (Custom Fields) on Servers.
server.delete Delete Server Allows the user to delete Servers from CloudBolt.
server.manage_credentials Manage Server Credentials Allows the user to view and manage the credentials used to execute remote Actions on Servers.
server.view_handler_specific_details View Handler-Specific Details Allows the user to see Resource Handler-specific details for Servers. E.g. the VMware Cluster, the AWS Region, etc.
server.all_actions Run All Server Actions Allows the user to execute any Server Action available on the Server.
server.view_power_schedule Manage Server's Power Schedule Allows the user to view the schedule for recurring power control on the Server.
server.manage_power_schedule Manage Server's Power Schedule Allows the user to edit the schedule for recurring power control on the Server.
server.add_disks Add Disks Allows the user to add disks to a Server.
server.remove_disks Remove Disks Allows the user to remove disks from a Server.
server.resize_disks Resize Disks Allows the user to resize disks on a Server.
server.resize Resize VM Change the type (size) of this Virtual Machine.
Resources
resource.view View Resources Allows the user to view Resources in CloudBolt.
resource.change_attributes Change Resource Attributes Change attributes for Resources in CloudBolt. 
resource.manage_parameters Manage Resource Parameters Add, remove, and modify parameters (Custom Fields) on Resources.
resource.all_actions Run All Resource Actions Allows the user to execute any Resource Action available on the Resource.
resource.view_power_schedule View Resource's Power Schedule Allows the user to view the schedule for recurring power control on the resource.
resource.manage_power_schedule Manage Resource's Power Schedule Allows the user to edit the schedule for recurring power control on the resource.
Groups
group.view View Group Details Allows the user to view details of a Group.
group.change_attributes Change Group Attributes Allows the user to change attributes of a Group.
group.delete Delete Group Allows the user to delete Groups.
group.create_subgroup Create Subgroup Allows the user to create a Subgroup within an existing Group.
group.manage_members Manage Group Memberships Allows the user to manage Group memberships.
group.manage_networks Manage Group Networks Allows the user to manage Group Networks.
group.manage_parameters Manage Group Parameters Allows the user to manage Group Parameters.
group.manage_subgroup_quota Manage Subgroup Quotas Allows the user to manage quotas for Subgroups of this Group.
group.manage_members_view_unassigned_usersManage Group Memberships - View Unassigned UsersAllows the User to view and add Users to Groups they manage that do not currently have a Group assigned to them (such as new Users). Requires group.manage_members permission to apply.
Blueprints
blueprint.manage Manage Blueprints
 Allows the user to perform most management actions on Blueprints that the Group can manage, such as setting the Resource Type, allowing additional Groups to manage or deploy it, adding Build Items, and many more. Exceptions provided instead by other permissions include the ability to configure the Blueprint so that any Group can deploy (blueprint.any_group_can_deploy) and the ability to create a new Parameter from the Blueprint (blueprint.create_parameters).
blueprint.any_group_can_deploy Configure Blueprints So Any Group Can Deploy Allows the user to modify the setting on a Blueprint that makes the Blueprint available for deployment by any and all Groups on the instance, both current and future Groups. This permission can have a broad impact. We recommend limiting this permission to Resource Admin or similar Role.
blueprint.create_parameters Create Blueprint Parameters Allows the user to create just Blueprint Parameters.
Jobs
job.view View Jobs Allows the user to view Jobs in CloudBolt.
job.view_logs View Job Logs Allows the user to view and download Job logs.
job.cancel Cancel Jobs Allows the user to cancel Jobs in progress.
job.rerun Re-run Jobs Allows the user to re-run failed Jobs.
Data Protection
dataprotection.manage_parameter_dataprotection_plan Manage Data Protection Plan Allows the user to manage servers' Data Protection plans.
dataprotection.restore_from_snapshot Restore Data Protection Snapshots Allows the user to restore a host from a Data Protection snapshot.
OneFuse
module.view View Modules Allows the user to view modules
policy.list List Policies Allows the user to list OneFuse Policies
policy.manage Manage Policies Allows the user to create, edit and delete OneFuse Policies
policy.execute Execute Policies Allows the user to execute OneFuse Policies
