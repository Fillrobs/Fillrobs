VMware

/api/v3/cmp/resourceHandlers/vmware/


Method Endpoint Name Description
GET /resourceHandlers/vmware/ Returns a list of VMware Resource Handlers.
POST /resourceHandlers/vmware/ Creates a VMware Resource Handler.
GET /resourceHandlers/vmware/{id}/  Returns the VMware Resource Handler for the given ID.
PATCH /resourceHandlers/vmware/{id}/ Update the VMware Resource Handler and returns the updated VMware.
DELETE /resourceHandlers/vmware/{id}/ Delete the specified VMware Resource Handler.
GET /resourceHandlers/vmware/{id}/clusters:remote/ Returns all Clusters available to import into the VMware Resource Handler. 
GET /resourceHandlers/vmware/{id}/clusters/ Lists all available Clusters in the VMware Resource Handler.
POST /resourceHandlers/vmware/{id}/clusters/ Adds a cluster to the VMware Resource Handler and returns the newly created cluster.
GET /resourceHandlers/vmware/{id}/networks:remote/
 Returns all Networks available to import into the VMware Resource Handler.
GET /resourceHandlers/vmware/{id}/networks/
 Lists all Networks available in the VMware Resource Handler.
POST /resourceHandlers/vmware/{id}/networks/ Adds a network to the VMware Resource Handler and returns the newly created network.
GET /resourceHandlers/vmware/{id}/templates:remote/
 Returns all templates available to import into the VMware Resource Handler.
GET /resourceHandlers/vmware/{id}/templates/
 Lists all templates available in the VMware Resource Handler.
POST /resourceHandlers/vmware/{id}/templates/
 Adds a template to the VMware Resource Handler and returns the newly created template.
POST /resourceHandlers/vmware/{id}/templates/{templateId}/parameters/  Adds credentials to a template to the VMware Resource Handler and returns the newly created template.
PUT /resourceHandlers/vmware/{id}/templates/{templateId}/parameters/ 
 Replace credentials to a template to the VMware Resource Handler and returns the template.
PATCH /resourceHandlers/vmware/{id}/templates/{templateId}/parameters/ 
 Update credentials to a template to the VMware Resource Handler and returns the updated template.
DELETE /resourceHandlers/vmware/{id}/templates/{templateId}/parameters/ 
 Delete credentials to a template to the VMware Resource Handler and returns the updated template.
POST /resourceHandlers/vmware/{id}/createServer:remote/
 Creates a server on the VMware Resource Handler and returns the job status.
GET /resourceHandlers/vmware/{id}/tasks:remote/{taskKey}/
 Returns the VMware Task for the given Task Key.
GET /resourceHandlers/vmware/{id}/servers:remote/{managedObjectId}/  Returns the VMware Virtual Machine for the given Managed Object ID.
POST /resourceHandlers/vmware/{id}/powerServer:remote/
 Perform power action on the VMware Resource Handler and returns status of action.
POST /resourceHandlers/vmware/{id}/modifyServer:remote/  Modify CPU and/or memory on the VMware Resource Handler and returns status of action.
POST /resourceHandlers/vmware/{id}/deleteServer:remote/  Deletes a server on the VMware Resource Handler and returns the job status.
POST
 /resourceHandlers/vmware/{id}/addDisk:remote/
 Adds a disk to a server on the VMware Resource Handler and returns the status of the disk add.
POST
 /resourceHandlers/vmware/{id}/extendDisk:remote/
 Extends a disk on a server on the VMware Resource Handler and returns the status of the extended disk.
POST
 /resourceHandlers/vmware/{id}/deleteDisk:remote/
 Delete a disk from a server on the VMware Resource Handler and returns the status of the disk delete.
POST
 /resourceHandlers/vmware/{id}/addNIC:remote/
 Add a NIC to a server on the VMware Resource Handler and returns the status of the NIC add.
POST
 /resourceHandlers/vmware/{id}/deleteNIC:remote/  Delete a NIC from a server on the VMware Resource Handler and returns the status of the NIC delete.
POST
 /resourceHandlers/vmware/{id}/createSnapshot:remote/
 Creates a snapshot of a server on the VMware Resource Handler and returns the snapshot ID.
POST
 /resourceHandlers/vmware/{id}/deleteSnapshot:remote/
 Deletes a snapshot of a server on the VMware Resource Handler and returns the snapshot ID.
POST
 /resourceHandlers/vmware/{id}/revertToSnapshot:remote/
 Deletes a snapshot of a server on the VMware Resource Handler and returns the snapshot ID.
