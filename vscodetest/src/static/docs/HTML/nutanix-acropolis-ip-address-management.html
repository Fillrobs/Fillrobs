IP Address Management for Nutanix Acropolis

There are two ways to manage IP Addresses in Nutanix Acropolis: managed networks and unmanaged networks.

Managed Networks with DHCP

With Managed networks, Acropolis will perform IP Address Management for VMs, assigning out IP Addresses via configurable DHCP pools.

Nutanix configuration

  1. Create a managed network in Nutanix Acropolis.

CloudBolt configuration

  1. Import the managed network on the Networks tab.
  2. Click the imported network. Configure the Addressing Schema to DHCP.
The template/image must be pre-configured for DHCP.

Unmanaged Networks

CloudBolt will perform IP Address Management for VMs, handing out IP addresses via configurable IP Pools or one of our IPAM integrations.

Nutanix Configuration

  1. Create an unmanaged network in Nutanix Acropolis.

CloudBolt Configuration

  1. Import the unmanaged network from the Networks tab.
  2. Click the imported network. Configure the network Addressing Schema to STATIC.
  3. Add the Parameter Skip Network Configuration. This will skip any network configuration done by the hypervisor and allow CloudBolt to send the cloud-init script.
  4. Add the Parameter Acropolis User Data for the cloud-init script.
  5. A cloud-init script is required for both Windows and Linux. There is a YAML sample script below. Or, create your own script. Use the Cloud Init docs to get started.
    The script below is a sample only. Please contact support@cloudbolt.io if you have issues.

    Copy Code
    YAML
    #cloud-config
    hostname: {{ server.hostname }}
    fqdn: {{ server.hostname }}.{{ server.sc_nic_0.dns_domain }}
     
    network:
      version: 1
      config:
        - type: physical
          control: auto
          name: eth0
          subnets:
             - type: static
               address: {{ server.sc_nic_0_ip }}
               netmask: {{ server.sc_nic_0.netmask }}
               gateway: {{ server.sc_nic_0.gateway }}
        - type: nameserver:
          address:
            - {{ server.sc_nic_0.dns1 }}
          search:
            - {{ server.sc_nic_0.dns_domain }}


  6. Choose a Static IP Management technology:
    • CloudBolt IP Pools 
    • An IP Address Management (IPAM) solution.
