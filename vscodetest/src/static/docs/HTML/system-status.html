System Status

Learn about the System Status page accessed from the Admin menu.

Introduction

The System Status page in CloudBolt displays a dashboard of Jobs and their progress, health checks for this CloudBolt instance, and allows a CB Admin user to enable admin-only mode.

Related Topics

  • Job Engine
  • Maintenance Mode
  • Infrastructure Health Checking with CIT

Prerequisites

  • Logged-on User must have the Super Admin Global Role enabled.

System Status User Interface

  1. To access the System Status user interface, click Admin > System > System Status.

Callout Feature Description
1 CloudBolt Mode
Click Enable admin-only mode to instantly log out all non-Admin Users if they attempt to access CloudBolt's UI. A message will appear at the top of all UI pages explaining that Admin-only Mode is enabled. Job will continue to run as normal, including recurring and one-time scheduled Jobs.
2

Job Engine

A snapshot of the Job Engine running processes show in the table below. Click Refresh to repopulate the table with fresh data.
3

System Status Colors

The status of running jobs will display here, along with a Runtime. Only processes running for 24 hours or fewer will display here. Jobs running for longer than 24 hours can be accessed in ShellPlus.

Green - Job Engine process has been running for less than 8 hours.

Yellow - Job Engine process has been running for 8-16 hours.

Orange - Job Engine process has been running for 16-24 hours. 

4

Jobs Graphs

 View bar graphs for Jobs in the last hour and Jobs in the last day, by Worker.
5 Health Checks This area provides an overview health check of this CloudBolt instance, including the Service checked and the current status.
