{# Prevent Chrome browser from offering to translate our UI #} {% comment %} The Windows group policy setting "Administrative Templates/Windows Components/Compatability View/Turn on Internet Explorer Standards Mode for Local Intranet" forces an IE browser mode of "Internet Explorer 7", which completely breaks our UI. Other factors can also trigger this behavior. This meta tag instructs IE to display with the latest browser mode, which is what we want because we're a modern site. {% endcomment %}
